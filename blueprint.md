# Lotto Number Generator

## Overview

A simple, stylish web application to generate and display random lotto numbers.

## Project Outline

### Styling (style.css)

*   **Theme:** Dark mode with a vibrant, modern aesthetic.
*   **Layout:** Centered content for a clean, focused user experience.
*   **Lotto Balls:** Circular elements with a gradient background and hover effect for interactivity.
*   **Button:** A prominent, styled button with a clear call-to-action.

### Structure (index.html)

*   **Container:** A main div to hold the application's content.
*   **Title:** A clear and descriptive heading.
*   **Number Display:** A container to hold the generated lotto balls.
*   **Button:** A button to trigger the number generation.

### Functionality (main.js)

*   **Number Generation:** A function to generate 6 unique random numbers between 1 and 45.
*   **Display:** A function to render the generated numbers as lotto balls in the UI.
*   **Event Handling:** An event listener on the "Generate Numbers" button to trigger the generation and display of new numbers.
*   **Initial Generation:** The application generates and displays a set of numbers on initial load.
